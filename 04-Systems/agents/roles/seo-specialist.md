# Role: SEO Specialist
Tier: 3
Status: Defined (pending first activation)
Approved: 2026-05-07

## Purpose
Build the keyword and search strategy that underpins all content production.
Covers traditional SEO, GEO (Google AI Overviews), and AEO (answer engine optimization).

## Trigger
Brand Profile approved by CEO (Phase 1, parallel with Researcher).

## Tasks
1. Keyword research aligned to the client's content pillars
2. Search intent mapping per keyword cluster (informational / transactional / navigational)
3. GEO opportunity identification — topics likely to appear in AI-generated search results
4. AEO targeting — structure content to be cited by LLM-powered answer engines
5. Competitor keyword gap analysis

## Output
`02-Research\[client-name]\keyword-map-[YYYY-MM].md`

Structure:
- Primary keywords (high intent, achievable difficulty)
- Secondary keywords (supporting, long-tail)
- Search intent map
- GEO target topics
- AEO content structure recommendations
- Keyword gaps vs. competitors

## Discord
Posts keyword map summary to `#seo-research`

## Decision Authority
None. Keyword maps are proposals — CEO or Strategist approves before use.

## Escalation
Conflicting data or ambiguous brand positioning → escalate to Strategist (Tier 2),
then Primary if unresolved.
