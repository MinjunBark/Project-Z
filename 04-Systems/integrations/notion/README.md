## Why We Use This
Technical documentation and user manuals — structured, searchable, shareable docs
hosted in Notion as a living knowledge base for Project Z.

## Setup
1. Create a Notion integration at https://www.notion.so/my-integrations
2. Copy the Internal Integration Token
3. Share the target Notion workspace/pages with the integration
4. Set NOTION_API_KEY in your .env file (see config-template.env)

## API / Connection Details
- Auth method: Internal Integration Token (Bearer)
- Key location: NOTION_API_KEY (env variable)
- Official docs: https://developers.notion.com/
